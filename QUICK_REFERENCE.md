## COMMON COMMANDS

```bash
# Install dependencies
pip install -r requirements.txt

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (Mac/Linux)
source venv/bin/activate

# Check setup
python verify.py

# Run app
streamlit run main.py

# Copy env file
cp .env.example .env  # Or: copy .env.example .env (Windows)

# Update requirements
pip freeze > requirements.txt
```

## TROUBLESHOOTING QUICK FIX

| Problem | Solution |
|---------|----------|
| API Key Error | Check `.env` file has GOOGLE_API_KEY |
| Import Error | Run `pip install -r requirements.txt` |
| Port in Use | `streamlit run main.py --server.port=8502` |
| App Won't Start | Run `python verify.py` |
| Slow Generation | Normal (30-60s) - AI is thinking |

## ERROR CODES & FIXES

```
Error: "API Key not found"
Fix: Add GOOGLE_API_KEY to .env

Error: "ModuleNotFoundError"
Fix: pip install -r requirements.txt

Error: "Connection refused"
Fix: Check internet connection

Error: "Port 8501 already in use"
Fix: streamlit run main.py --server.port=8502
```
