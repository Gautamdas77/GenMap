## 💡 HOW IT WORKS

### User Journey
```
1. User opens app
   ↓
2. Enters topic (e.g., "Web Development")
   ↓
3. Sets timeline (e.g., "6 Months")
   ↓
4. Selects learning level
   ↓
5. Clicks "Generate Roadmap"
   ↓
6. AI creates personalized roadmap
   ↓
7. App displays modules, quizzes, projects
   ↓
8. User explores and learns!
```

```
Streamlit UI
   ↓

   ↓

   ↓
```
